# Tools

## Emacs

Set GUI font.
```lisp
(add-to-list 'default-frame-alist '(font . "Monaco"))
```

Set file format.
```lisp
(set-buffer-file-coding-system 'unix)
```

Kill to the start of line.
```
C-u 0 C-k
```
